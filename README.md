# Correos-MOTOSTUDENT
Código y archivos de la aplicación para gestionar el envío de correos a los distintos departamentos del proyecto MOTOSTUDENT
----------------------------------------------------------------------------------------------------------------------------
CONTENIDO:
-Fichero principal que contiene la interfaz y la mayoría del programa
-Fichero auxiliar que contiene la función empleada para enviar los correos
-Ejecutable(sin necesidad de tener python) para windows 10
----------------------------------------------------------------------------------------------------------------------------
CONFIGURACIÓN INICIAL PREVIA:
-En la ubicación donde se guarde y use el ejecutable(o se ejecute el .py principal):
  -Crear un archivo de texto(bloc de notas) llamado: Credenciales.txt dentro de este:
    -En la primera linea introducir la dirección de correo electrónico(completa) que se va a emplear para enviar los correos.
    -En la segunda linea introducir la contraseña de la cuenta de dicho correo electrónico.
  -Habilitar desde google accounts(o la página correspondiente) la posibilidad de que aplicaciones de terceros accedan a la cuenta
  -Crear una carpeta llamada: listasDeCorreo, dentro de esta:
    -Crear un archivo de texto(bloc de notas(extensión .txt) para cada departamento utilizando los siguientes nombres(sin los guiones) para cada archivo:
      -CFD
      -Comunicación y empresas
      -Estructural y dinámica y suspensiones
      -IT
      -Power train electric
      -Power train petrol
      -Producción
      -Proyectos
    -Dentro de cada uno de los archivos creados en la carpeta, introducir las direcciones de correo electrónico de los integrantes de cada departamento,
    en el archivo en cuestión, introducir una sola dirección de correo por linea.
--------------------------------------------------------------------------------------------------------------------------------
GUÍA DE USO:
-Realizar la configuración previa ya mencionada.
-Ejecutar el ejecutable o el archivo correoDepartamentos.py
De ahí en adelante la interfaz es muy intuitiva y explica todo
